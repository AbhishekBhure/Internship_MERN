# INTERNSHIP

MERN PROJECT:

To run the project split terminal

##

Terminal1
command:

```bash
  cd frontend
```

```bash
  npm start
```

Terminal 2 command:

```bash
  npm run dev
```

Add proxy in package.json present in frontend folder  
 "proxy": "http://192.168.139.42:4000"
